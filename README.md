## Introdução

A multiplicação de matrizes é uma operação fundamental na Computação, amplamente utilizada em diferentes áreas. O algoritmo clássico apresenta complexidade O(n³), enquanto o algoritmo de Strassen aplica o paradigma de divisão e conquista para reduzir o número de multiplicações e melhorar o desempenho assintótico.

Este projeto foi desenvolvido para a disciplina **Algoritmos e Complexidade Computacional** da **Universidade Federal do Ceará (UFC)** e apresenta uma comparação prática entre essas duas abordagens por meio de benchmarks.

## Objetivo

O objetivo deste projeto é comparar o desempenho do algoritmo clássico de multiplicação de matrizes com o algoritmo de Strassen, analisando o impacto do uso de divisão e conquista e identificando em quais cenários o método de Strassen se mostra mais vantajoso.
